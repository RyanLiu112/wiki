    动态规划¶

    能用动态规划解决的问题，需要满足三个条件：最优子结构，无后效性和子问题重叠。

    DP 步骤：

    • 确定 dp 数组以及下标的含义（dp[i][j], dp[i]）
    • 确定递推公式
    • dp 数组如何初始化（全0/1、开头1后面0）
    • 确定遍历顺序
    • 举例推导 dp 数组

    DP 基础¶

    509.斐波那契数¶

    https://leetcode.cn/problems/fibonacci-number/

    https://programmercarl.com/0509.%E6%96%90%E6%B3%A2%E9%82%A3%E5%A5%91%E6%95%B0.html

    class Solution:
    def fib(self, n: int) -> int:
        if n <= 1:
            return n
        dp = [0 for i in range(n + 1)]
        dp[0], dp[1] = 0, 1
        for i in range(2, n + 1):
            dp[i] = dp[i - 2] + dp[i - 1]
        return dp[n]

    class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2:
            return n
        dp = [0 for i in range(n + 1)]
        dp[1], dp[2] = 1, 2
        for i in range(3, n + 1):
            dp[i] = dp[i - 2] + dp[i - 1]
        return dp[n]

    746.使用最小花费爬楼梯¶

    https://leetcode.cn/problems/min-cost-climbing-stairs/

    https://programmercarl.com/0070.%E7%88%AC%E6%A5%BC%E6%A2%AF.html

    class Solution:
    def minCostClimbingStairs(self, cost: List[int]) -> int:
        n = len(cost)
        dp = [0 for i in range(n + 1)]
        for i in range(2, n + 1):
            dp[i] = min(dp[i - 2] + cost[i - 2], dp[i - 1] + cost[i - 1])
        return dp[n]

    62.不同路径¶

    https://leetcode.cn/problems/unique-paths/

    https://programmercarl.com/0062.%E4%B8%8D%E5%90%8C%E8%B7%AF%E5%BE%84.html

    解题思路¶

    1. 确定 dp 数组以及下标的含义

    dp[i][j] 表示从 (0, 0) 出发，到达 (i, j) 处的路径数量

    1. 确定递推公式

    由于机器人只能向右和向下走，因此，到达 (i, j) (i > 1, j > 1) 可以从 (i - 1, j) 向下走或者 (i, j - 1) 向右走，所以：\(dp[i][j] = dp[i - 1][j] + dp[i][j - 1]\)

    1. dp 数组如何初始化

    dp[i][0] 和 dp[0][j] 都是1

    1. 确定遍历顺序

    从左到右一层一层遍历

    1. 举例推导 dp 数组

    略。

    代码

    class Solution:
    def uniquePaths(self, m: int, n: int) -> int:
        dp = [[0 for j in range(n)] for i in range(m)]
        for j in range(n):
            dp[0][j] = 1
        for i in range(m):
            dp[i][0] = 1
        for i in range(1, m):
            for j in range(1, n):
                dp[i][j] = dp[i - 1][j] + dp[i][j - 1]
        return dp[m - 1][n - 1]

    63.不同路径 II¶

    https://leetcode.cn/problems/unique-paths-ii/

    https://programmercarl.com/0063.%E4%B8%8D%E5%90%8C%E8%B7%AF%E5%BE%84II.html

    class Solution:
    def uniquePathsWithObstacles(self, obstacleGrid: List[List[int]]) -> int:
        m, n = len(obstacleGrid), len(obstacleGrid[0])
        dp = [[0 for j in range(n)] for i in range(m)]
        # 不能写 dp = [[0] * n] * m，会有地址的问题
        for j in range(n):
            if obstacleGrid[0][j] == 1:
                break
            else:
                dp[0][j] = 1
        for i in range(m):
            if obstacleGrid[i][0] == 1:
                break
            else:
                dp[i][0] = 1
        for i in range(1, m):
            for j in range(1, n):
                if obstacleGrid[i][j] != 1:
                    dp[i][j] = dp[i - 1][j] + dp[i][j - 1]
        return dp[m - 1][n - 1]

    343.整数拆分¶

    https://leetcode.cn/problems/integer-break/

    https://programmercarl.com/0343.%E6%95%B4%E6%95%B0%E6%8B%86%E5%88%86.html

    class Solution:
    def integerBreak(self, n: int) -> int:
        if n <= 3:
            return n - 1
        dp = [0 for i in range(n + 1)]  # dp[i] 表示正整数i拆分后各个因数的最大乘积
        dp[2] = 1
        dp[3] = 2
        for i in range(4, n + 1):
            for j in range(1, i):
                dp[i] = max(dp[i], (i - j) * j, dp[i - j] * j)
        return dp[n]

    416.分割等和子集¶

    https://leetcode.cn/problems/partition-equal-subset-sum/

    https://programmercarl.com/0416.%E5%88%86%E5%89%B2%E7%AD%89%E5%92%8C%E5%AD%90%E9%9B%86.html

    class Solution:
    def canPartition(self, nums: List[int]) -> bool:
        s = sum(nums)
        if s % 2:
            return False
        m, n = len(nums), s // 2
        dp = [[0 for j in range(n + 1)] for i in range(m)]  # dp[i][j]表示只允许放0-i个物品、背包容量为j时，背包能达到的最大价值。
        for j in range(n):
            if j >= nums[0]:
                dp[0][j] = nums[0]
        for i in range(1, m):
            for j in range(1, n + 1):
                if j < nums[i]:
                    dp[i][j] = dp[i - 1][j]
                else:
                    dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - nums[i]] + nums[i])
        return dp[m - 1][n] == n

    1049.最后一块石头的重量 II¶

    https://leetcode.cn/problems/last-stone-weight-ii/

    https://programmercarl.com/1049.%E6%9C%80%E5%90%8E%E4%B8%80%E5%9D%97%E7%9F%B3%E5%A4%B4%E7%9A%84%E9%87%8D%E9%87%8FII.html

    class Solution:
    def lastStoneWeightII(self, stones: List[int]) -> int:
        s = sum(stones)
        m, n = len(stones), s // 2
        dp = [[0 for j in range(n + 1)] for i in range(m)]
        for j in range(1, n + 1):
            if j >= stones[0]:
                dp[0][j] = stones[0]
        for i in range(1, m):
            for j in range(1, n + 1):
                if j < stones[i]:
                    dp[i][j] = dp[i - 1][j]
                else:
                    dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - stones[i]] + stones[i])
        return s - 2 * dp[m - 1][n]
